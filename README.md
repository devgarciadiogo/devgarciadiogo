### 🚌 [ReclamaBus](https://github.com/devgarciadiogo/reclamabus)
> Sistema de ocorrências em transportes públicos do município de Niterói/RJ

Projeto acadêmico desenvolvido em equipe que permite aos usuários relatar incidentes em linhas de ônibus da cidade — desde problemas com ar-condicionado e higiene até relatos de assaltos e reclamações sobre motoristas. O objetivo é oferecer uma plataforma transparente para que cidadãos compartilhem informações relevantes sobre o transporte público.

### 💰 [Finance App](https://github.com/devgarciadiogo/finance-app)
> Sistema de gestão financeira pessoal — em evolução contínua

Projeto pessoal que nasceu em Java puro e está sendo evoluído progressivamente até uma **API REST completa com Spring Boot e autenticação JWT**. Um laboratório prático onde aplico os conceitos que estudo.

**Fase atual — funcionalidades implementadas:**
- ✅ Cadastro de receitas e despesas por categoria
- ✅ Cálculo automático de saldo
- ✅ Relatório com maior despesa e receita do período
- ✅ Agrupamento de gastos por categoria com Streams
- ✅ Metas de gasto por categoria com alertas
- ✅ Validações e exceções customizadas

**Roadmap:**
- 🔜 Persistência em JSON
- 🔜 Banco de dados PostgreSQL
- 🔜 API REST com Spring Boot
- 🔜 Autenticação com JWT
